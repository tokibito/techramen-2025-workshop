概要
----

このワークショップは、TechRAMEN 2025カンファレンスで開催される「SQL基礎トレーニング～JOIN、GROUP BYの使い方を学ぼう」の教材です。

SQLの基本的な構文から始めて、実践的なJOIN操作とGROUP BY句の使い方を学習します。

対象者
------

* SQLをこれから学びたい方
* JOINやGROUP BYの使い方に自信がない方
* 実践的なSQLクエリを書けるようになりたい方

学習目標
--------

このワークショップを完了すると、以下のことができるようになります：

1. 基本的なSQLクエリ（SELECT、INSERT、UPDATE、DELETE）を記述できる
2. 複数のテーブルを適切にJOINできる
3. GROUP BYを使用して集計処理ができる
4. 実践的なデータ分析クエリを作成できる

必要な環境
----------

* Docker Desktop（またはDocker Engine + Docker Compose）
* Webブラウザ（pgAdmin用）
* テキストエディタ
